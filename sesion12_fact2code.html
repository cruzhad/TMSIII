📚 Teoría: Factorización de Trinomios

📓 Copia en tu cuaderno

1. ¿Qué es factorizar?

Factorizar es escribir una expresión como producto de sus factores. Es el proceso inverso a multiplicar.

Multiplicar: $(x+3)(x+4) = x^2 + 7x + 12$

Factorizar: $x^2 + 7x + 12 = (x+3)(x+4)$

2. Método para factorizar x² + bx + c:

Buscar dos números $m$ y $n$ tales que:

  • $m + n = b$ (suma = coeficiente de x)
  • $m × n = c$ (producto = término independiente)

Entonces: $x^2 + bx + c = (x + m)(x + n)$

Ejemplo 1: Factorizar $x^2 + 5x + 6$

Paso 1: Identificar b = 5 y c = 6

Paso 2: Buscar dos números que sumen 5 y multipliquen 6

Intentos: 1 y 6 → suma 7 ✗

2 y 3 → suma 5 ✓ y producto 6 ✓

Paso 3: $x^2 + 5x + 6 = (x + 2)(x + 3)$

Verificación: $(x+2)(x+3) = x^2 + 3x + 2x + 6 = x^2 + 5x + 6$ ✓

Ejemplo 2: Factorizar $x^2 - 7x + 10$

Paso 1: b = -7, c = 10

Paso 2: Necesitamos dos números que sumen -7 y multipliquen 10

Como el producto es positivo, ambos deben tener el mismo signo

Como la suma es negativa, ambos son negativos

-2 y -5 → suma -7 ✓ y producto 10 ✓

Paso 3: $x^2 - 7x + 10 = (x - 2)(x - 5)$

Ejemplo 3: Factorizar $x^2 + 2x - 15$

Paso 1: b = 2, c = -15

Paso 2: Dos números que sumen 2 y multipliquen -15

Como el producto es negativo, tienen signos diferentes

5 y -3 → suma 2 ✓ y producto -15 ✓

Paso 3: $x^2 + 2x - 15 = (x + 5)(x - 3)$

3. Reglas de signos:

  • Si c > 0: los dos números tienen el mismo signo (ambos + o ambos -)
  • Si c < 0: los dos números tienen signos diferentes
  • El signo de b indica cuál número tiene mayor valor absoluto

4. ¿Para qué sirve?

La factorización es esencial para resolver ecuaciones cuadráticas, simplificar fracciones algebraicas, encontrar raíces de funciones, optimizar expresiones y modelar situaciones reales.
