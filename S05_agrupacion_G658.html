📐 Signos de Agrupación — Reglas de eliminación

( )
Paréntesis — nivel más interno. Se resuelven siempre primero.
[ ]
Corchetes — nivel medio. Se resuelven cuando el paréntesis ya fue eliminado.
{ }
Llaves — nivel exterior. Se resuelven al final.

Reglas de cambio de signo al eliminar

+(a − b + c)  =  a − b + c    (signo + NO cambia nada)
−(a − b + c)  =  −a + b − c    (signo − CAMBIA TODOS)
Anidación:   { [ ( ) ] }  →  resuelve de adentro hacia afuera

Ejemplo con x — cómo documentarlo en el cuaderno

Expresión: 3·[x² − (2x + 1)] + 4   con x = 3
Paso 1 (paréntesis): (2·3 + 1) = 7
Paso 2 (corchete): [9 − 7] = 2
Paso 3 (multiplicación): 3·2 = 6
Paso 4 (suma): 6 + 4 = 10
